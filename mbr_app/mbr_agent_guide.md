# Keel MBR — Monthly Business Review Automation (Built into Keel)

## Overview
MBR (Monthly Business Review) report generation is a **built-in capability of the Keel agent**.
When a user explicitly asks to run the MBR for a given month, **Keel generates it directly** —
no separate agent required. This keeps everything in one agent that ships together when Keel
is published/updated.

- **Lineage:** Knowledge originated from the Fola report agent; folded directly into Keel.
- **Trigger:** Explicit only — "run the MBR for <Month> <Year>"
- **Model:** Keel's model (claude-4-6-opus)
- **Backup agent:** `~/.code_puppy/agents/keel-mbr.json` is kept on disk as a standalone
  backup but is NOT the primary path (Keel does MBRs itself).

## Architecture
```
User → Keel: "run the MBR report for June 2026"
         ↓ (Keel's built-in MBR capability — no handoff)
       Keel: queries BigQuery live, drafts insights,
             asks user for non-BQ inputs, builds branded HTML
         ↓
       Returns report path / share link → user iterates
```

Keel has all tools needed (BigQuery, create_file/replace_in_file, shell,
universal_constructor) to execute end-to-end.

## Month Parameterization
When asked for `<Month> <Year>`:
- **Current** = first→last day of that month
- **Prior** = preceding month (MoM)
- **LY** = same month previous year (YoY, where available)
- All metrics computed **live** from BigQuery — never hardcoded.

## Core Data & Methodology Locks (inherited)
- Table: `wmt-site-content-strategy.scs_production.hp_summary_asset` (`session_start_dt`)
- Click/CTR/CPTS/ATC/GMV: **ex-IIC AND ex-Social** →
  `IFNULL(moduletype,'') NOT IN ('InteractiveImageCarousel','socialCarouselV2')`
- Activations: **with-IIC** (no moduletype filter) — scorecard alignment
- ATF = `content_zone IN ContentZone1..6`; BTF = ContentZone7+ or null
- Always `SAFE_DIVIDE()`; always date-filter first
- Sessions denominator: `hp_session`, `traffic_source_lvl2='Organic: Direct'`

## MBR Structure (director-approved, 7 sections)
1. Executive Summary
2. Executive Roadmap Review
3. Business Performance — Headlines
4. Customer Cohorts Strategy (two-engine framework)
5. HPOV Deep Dive
6. SIG ATF Deep Dive
7. Appendix — Definitions

## Insights = Mix Mode
keel-mbr auto-drafts narrative drivers from the data, then surfaces them to the
user to confirm/adjust qualitative context (events, launches, roadmap) and to
supply external numbers (W+ sign-up rate, ad revenue, sitewide traffic, SOV bps)
before final publish.

## Report Generation Engine
A proven reference generator lives at:
`~/Library/CloudStorage/OneDrive-WalmartInc/Desktop/clickfather/may_mbr_v5.py`

keel-mbr generalizes this for any month by:
1. Reading the engine to reuse helpers (`pch`, `fnum`, `fdol`, `fbadge`, `bar_chart`)
2. Creating a new `<month>_mbr.py` with the same structure, populated from live BQ
3. Running it to emit `<month><year>_nav_mbr_v{N}_{timestamp}.html`
4. Opening + iterating (expect 4–6 director-style edit cycles)

Brand colors: `#0071CE` blue · `#FFC220` yellow · `#76C043` up · `#E31837` down.
